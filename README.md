## Installation

[Node.js](https://nodejs.org/) v16+.

```sh
git clone https://github.com/Robertotheto/ignite-lab-01.git

cd ignite-lab-01

docker-compose up -d

cd purchases
npm i
npm run start:dev

cd classroom
npm i
npm run start:dev

cd gateway
npm i
npm run start:dev

```
